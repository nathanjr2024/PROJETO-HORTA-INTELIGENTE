# Partes Interessadas (Stakeholders)

**Projeto:** Horta Inteligente

---

## Quem usa o sistema

- **Usuário principal:** A pessoa responsável pela horta, que acessa o site para ver os dados dos sensores e saber se a horta foi regada.
- **Cliente:** O professor orientador do PI, que valida se o sistema funciona corretamente.

## Quem instala e mantém

- **Instalador:** Um membro da equipe que conecta os sensores ao microcontrolador e instala o dispositivo na horta.
- **Manutenção:** A própria equipe, que resolve problemas de hardware ou atualiza o código quando necessário.

## Quem pode causar problemas

- **Atacante:** Alguém que tente acessar o site sem permissão para ver os dados ou acionar a irrigação indevidamente. Por isso o site precisa de login.
