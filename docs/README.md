# Horta Inteligente — Projeto Integrador

**Equipe:** Nathan Junior, Bruno Avelino e Kauã Martins  
**Tema:** Sistema para monitorar e regar uma horta automaticamente  
**Escopo:** Um dispositivo com sensores instalado na horta envia dados (temperatura, umidade do solo e nível de água) para um site, que mostra essas informações e aciona a irrigação automaticamente quando necessário.
